## Visualizar:
 - Primeiro clone o repositório:
    - ``` git clone git@github.com:T-abran/personal-crud.git ```
 - Entre no repositório: 
   - ``` cd personal-crud ```
    
 - Entre na pasta front:
   - ``` cd front ```
 - Inicia o projeto:
    - ``` npm start ```
  
 - Entre na pasta back:
   - ``` cd back ```
 - Inicia o projeto:
    - ``` npm run dev ```
